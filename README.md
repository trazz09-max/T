# Tesla Stream Hub

A large-button streaming website designed for use in a Tesla browser while the vehicle is parked.

## Included

- Large touchscreen-friendly streaming buttons
- Search
- Favorite apps saved in the browser
- Custom links saved in the browser
- Responsive layout for Tesla screens, tablets, and phones
- Parked-use and compatibility notice

## Preview locally

Open `index.html` in a browser.

Some browsers limit local-storage features when opening a file directly. For the most reliable preview, run a small local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Publish it free

### GitHub Pages
1. Create a GitHub repository.
2. Upload `index.html`, `styles.css`, and `app.js`.
3. Open repository Settings → Pages.
4. Select the main branch and root folder.
5. Open the generated website address in the Tesla browser and save it as a favorite.

### Netlify Drop
1. Visit Netlify Drop on a computer.
2. Drag the entire folder onto the page.
3. Netlify creates a public website address.
4. Open that address in the Tesla browser.

## Important limitations

This website is only a launcher. It does not bypass subscriptions, account logins, DRM, browser compatibility rules, Tesla safety restrictions, or streaming-service policies. Playback availability will vary by service and vehicle software version.
